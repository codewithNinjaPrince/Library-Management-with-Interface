# LibTrack API Platform

LibTrack is a Smart Library & Book Inventory Management System designed to manage books, members, borrowing operations, inventory tracking, and analytics through REST APIs.

The platform helps educational institutes and libraries digitize their book management process and replace manual Excel-based workflows.

## Features

- Book Management
- Member Management
- Borrow & Return System
- Inventory Tracking
- JWT Authentication
- CSV Book Upload
- Reading Analytics
- Audit Logs
- REST APIs
- Swagger API Documentation

## Tech Stack

- Python
- FastAPI
- MongoDB Atlas
- JWT Authentication

## Purpose of the Project

This project was built to solve common library management problems such as:

- Duplicate records
- Missing books
- No borrowing history
- Difficult report generation
- Lack of centralized APIs
